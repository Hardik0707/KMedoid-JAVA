# KMedoid-JAVA
Source Code for KMedoid Algorithm in JAVA    
This is a basic implementation of KMedoid Algorithm having cluster size = 2   

# Run Code

$ javac KMedoid.java  
$ java KMedoid  

Sample Inputs:    
  
No. of Clusters = 2  
Enter Number of data: 10    
Enter X and Y values of data
